## Installation

Using [lazy.nvim](https://github.com/folke/lazy.nvim) or your preferred package manager:

```lua
{ "drewxs/ash.nvim" }
```

## Configuration

Options can be found in [`lua/ash/options.lua`](lua/ash/options.lua).
